# GoogleTranslate
A simple project to use google translate api for free

### How To Use?
See some examples :
```php
<?php
  require 'GoogleTranslate.php';
  $gtr = new GoogleTranslate();
  echo $gtr->translate('Test', 'en');
?>
```
